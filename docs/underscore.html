Desarrollador por Jeremy Ashkenas, creador de Backbone y Coffee scripts
Es un conjunto de utilidades para manipular colecciones y arrays
Tiene un potente motor de templates basado en el jquery.tmpl de John Resig
Principales funciones:
_.each([1, 2, 3], alert);

var even = _.find([1, 2, 3, 4, 5, 6], function(num){ return num % 2 == 0; });

_.where(listOfPlays, {author: "Shakespeare", year: 1611})
Templating
Es muy similiar a la sintáxis de ASP.NET
Se pueden invocar funciones de Javascript desde el template

<div id="<%-id%>">
<label>Nombre</label><span><%-usuario.nombre%></span>
<label>Es mayor de edad: <%-calcularEdad(usuario.edad)%> </label>
</div>
