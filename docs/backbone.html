Desarrollador por Jeremy Ashkenas, creador de Underscore y Coffee scripts
Es un framework para desarrollar aplicaciones single page
Se base en el patrón MVC pero no lo respeta totalmente (no existen controllers)
No provee lineamientos sobre cómo organizar la arquitectura
Provee un conjuto de elementos pre-definidos pensados para interactuar entre sí
Se puede usar cualquier de sus componentes independientemente del resto
Es muy liviano


Se base en underscorejs (del mismo autor)
