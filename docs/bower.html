Es un manejador de paquetes (como Nuget o npm) basado en GIT

Para utilizarlos basta tener dos archivos en nuestro directorio:

    
	
  • .bowerrc para indicar parámetros de configuración, por ejemplo el directorio destino
    • 
	
  • bower.json para indicar los paquetes que queremos instalar
    •
Ejemplo de archivos de configuración
.bowerrc
{
	"directory": "js/vendor"
}

bower.json

{
	"name": "aplicacion",
	"version": "0.1",
	"dependencies":
		{ 
			"jquery": "1.11.1",
			"underscore": "1.6.0"
		}
}
uso:
bower install
