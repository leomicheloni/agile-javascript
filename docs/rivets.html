RIVETS.JS
 
Permite realizar binding automático, es decir, 
tener un model en nuestra aplicación y actualizar la UI automáticamente a partir de mismo y viceversa.
Es muy extensible
Es muy liviano
Ejemplo:
var model = {
		nombre: "Michael",
		apellido: "Richards",
		saludar: function(){
			return "Hola, mi nombre es " + this.nombre;
		}	
	};
var view = Rivets.bind($("#vista"), model);

Vista
<div>
<span rv-text="nombre"></span>
<span rv-text="apellido"></span>
<input type="button" value="saludar" rv-on-click="saludar" />
</div>

Permite utilizar filtros sobre los valores que se aplican a la vista.
Podemos definir nuestros propiedos adapters si queremos variar la forma de relacionarse con el modelo, por ejemplo, para usar Backbone.
